# propertyweb (frontend)

Next.js + Tailwind v4 + shadcn/ui.

## Run

```bash
npm run dev
```

Open `http://localhost:3000`.

The project includes:

- Public pages: `/`, `/listings`, `/property/[slug]`, `/localities`, `/locality/[slug]`, `/type/[slug]`, `/sell`
- Seller shell: `/seller/*`
- Admin shell: `/admin/*`
